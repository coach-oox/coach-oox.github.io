```html
<div class="flash"></div>
```

```css
.flash {
  animation: goAway 0.5s ease-in-out forwards;
  animation-delay: 2s;
}

@keyframes goAway {
  from {
    transform: none;
    opacity: 1;
  }
  to {
    transform: tanslateY(-50px);
    opacity: 0;
  }
}
```
