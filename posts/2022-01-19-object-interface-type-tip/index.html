초보 개발자를 위한 VSCode 단축키 기초 활용 TIP 3가지

Dohae Lee on Jan 192022-01-19T11:44:24+08:00
development
2 min read

복잡한 API 데이터 타입을 지정하는 방법

  1. 받아오려는 데이터를 콘솔에 찍어 봄
  2. 마우스 오른쪽 클릭 → Store object as global variable
  3. 콘솔에서 Object.keys(temp1).join()하면 객체의 모든 키 값 가져올 수 있음
  4. 해당 키를 앞뒤 따옴표 없이 복사
  5. 쉼표 하나 선택 후, cmd + d 단축키 사용해서 모든 쉼표 선택해 지우고 엔터
  6. 모든 키 앞에 커서가 깜빡이는 상태에서 cmd + → 단축키로 끝으로 이동한 다음, :와 ; 작성
  7. 콜론과 세미콜론 사이에 커서가 깜빡이는 상태로 놔둠
  8. 콘솔에서 Object.values(temp1).map((value) => typeof value).join()하면 객체의 모든 값의 타입을 가져올 수 있음
  9. 해당 값을 앞뒤 따옴표 없이 복사
  10. 인터페이스 위쪽 빈 공간에 붙여넣고 동일한 방법으로 쉼표 제거 후 엔터
  11. 모든 값 앞에 커서가 깜빡이는 상태에서 cmd + shift + → 단축키로 각각을 복사해 잘라내기
  12. 깜빡이는 빈 커서 그대로 인터페이스 키 값 앞으로 이동
  13. cmd + → 단축키로 끝으로 이동한 다음, 콜론과 세미콜론 사이에 붙여넣기

이때 주의할 점은, 최종적으로 object로 표기되는 타입의 경우 (사실은 오브젝트들의 배열이라던지) 정확하지 않을 수 있다. 따라서, 필요하지 않다면 지워버리는 것이 좋고, 필요하다면 각각의 타입을 인터페이스로 만들어 수정해줘야 한다.
