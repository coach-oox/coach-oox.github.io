입문자를 위한 개발이 빨라지는 VSCode 단축키 11가지

Dohae Lee on Jan 212022-01-21T00:00:00+08:00
development
4 min read

강의에 질문이 들어와서 겸사겸사 해보는 단축키 정리. 편의를 위해 Windows 단축키도 작성했으나 MacOS를 기준으로 작성했기 때문에 혹시 틀린 부분이 있을 수 있어 다음 링크를 함께 참고하면 좋다.

  • MacOS Keyboard Shortcuts
  • Windows Keyboard Shortcuts

여러 줄 동시에 선택

#1 option (alt) + click

option (Windows는 alt) 키를 누른 채로 마우스를 클릭하면, 클릭하는 곳마다 마우스 커서가 활성화 되어, 여러 줄을 동시에 선택할 수 있다.

같은 단어 선택

1
2
3
4
5

hello.world();
hello.world();
hello.world();
hello.world();
hello.world();

#2 cmd (ctrl) + d

위와 같은 코드를 가정할 때, 선택하고자 하는 단어를 드래그해서 블록을 잡고, cmd (Windows는 ctrl) + d 키를 누르면, 키를 한 번 누를 때마다 하나씩 선택한 단어와 동일한 단어가 추가로 선택된다. 즉, 라인 1의 hello를 블록 잡고 cmd + d를 한 번 누르면 라인 2의 hello가 선택되고, 추가로 누를 때마다 라인 3, 라인 4 그리고 라인 5의 hello가 순서대로 선택되게 된다.

#3 cmd (ctrl) + shift + l

선택하고자 하는 단어를 드래그해서 블록을 잡고, cmd (Windows는 ctrl) + shift + l 키를 누르면, 현재 선택된 단어와 동일한 단어가 모두 선택된다.

문서 내 이동

#4 cmd (ctrl) + 상하 방향키

cmd + 상하 방향키 (Windows는 ctrl + home / end)를 사용하면 문서의 처음과 끝으로 바로 이동할 수 있다.

#5 cmd (ctrl) + 좌우 방향키

cmd (Windows ctrl) + 좌우 방향키를 사용하면, 현재 커서가 위치한 라인의 처음과 끝으로 바로 이동할 수 있다.

문서 내 선택

#6 cmd (ctrl) + shift + 상하 방향키

cmd (Windows는 ctrl) + shift + 상하 방향키를 사용하면, 현재 커서 위치부터 문서의 처음 또는 끝까지 한 번에 선택할 (블록을 잡을) 수 있다.

#7 cmd (ctrl) + shift + 좌우 방향키

cmd (Windows는 ctrl) + shift + 좌우 방향키를 사용하면, 현재 커서 위치부터 현재 라인의 처음 또는 끝까지 한 번에 선택할 수 있다.

#8 option (alt) + cmd (ctrl) + 상하 방향키

option + cmd (Windows는 alt + ctrl) + 상하 방향키를 사용하면, 현재 커서부터 위 또는 아래 라인들을 함께 선택할 수 있다.

#9 option (alt) + shift + 좌우 방향키

option (Windows는 alt) + shift + 좌우 방향키를 누르면, 현재 커서 위치에서 단어 단위만큼씩 이동하며 추가로 블록이 잡힌다. (추가로 cmd 또는 ctrl을 함께 누른다면 한 글자씩 이동한다.)

현재 라인 이동

#10 option (alt) + 상하 방향키

특정 라인에 커서를 두고 option (Windows는 alt) + 상하 방향키를 누르면, 해당 라인이 위 또는 아래로 이동한다. 이때 이동하려는 위치에 이미 다른 글자가 있더라도 Replace 된다.

현재 라인 복사

#11 option (alt) + shift + 상하 방향키

특정 라인에 커서를 두고 option (Windows는 alt) + shift + 상하 방향키를 누르면, 위 또는 아래로 현재 라인이 복사 및 붙여넣기 된다.
