타입스크립트에서 styled-components 사용하기 (Theme, Props)

Props 타입 지정

타입스크립트를 사용할 때 styled-components 컴포넌트에서 사용하는 Props를 체크하기 위해서는 다음과 같이 Props에 대한 interface를 작성한 뒤, styled.element<interface> 형식으로 Props 타입을 지정한다.

1
2
3
4
5
6
7

interface IContainerProps {
  bgColor: string;
}

const Container = styled.div<IContainerProps>`
  background-color: ${(props) => props.bgColor};
`;

Theme 사용법

  • Create a declarations file

Declaration 작성

먼저 Declaration 파일을 작성해 줘야 한다. styled.d.ts와 같이 .d.ts 파일을 생성하고, 다음 코드 예제와 같이 사용하려는 테마의 인터페이스를 작성한다. 이 부분을 제외하면, 기존에 자바스크립트 환경에서 테마를 사용하는 방법과 동일하게 진행하면 된다. styled-components에서 테마를 사용하는 방법 이 글에서도 다루었다시피 하위 내용과 같다.

1
2
3
4
5
6
7
8

import "styled-components";

declare module "styled-components" {
  export interface DefaultTheme {
    textColor: string;
    bgColor: string;
  }
}

Theme 작성

다음으로는 테마 파일을 작성한다. my-theme.ts와 같은 파일을 생성하고, 다음 코드 예제와 같이 정의한 인터페이스를 사용하는 (예제에서는 DefaultTheme) 테마를 작성한다.

1
2
3
4
5
6

import { DefaultTheme } from "styled-components";

export const theme: DefaultTheme = {
  bgColor: "white",
  textColor: "black",
};

ThemeProvider

styled-components에서 제공하는 ThemeProvider 컴포넌트를 사용해서 테마를 사용할 애플리케이션 (또는 컴포넌트)을 감싼다. 이때 ThemeProvider는 다음 코드 예제와 같이 테마 파일을 값으로 가지는 theme이라는 필수 Prop이 요구된다.

import React from "react";
import ReactDOM from "react-dom";
import App from "./App";
import { ThemeProvider } from "styled-components";
import { theme } from "./theme";

ReactDOM.render(
  <React.StrictMode>
    <ThemeProvider theme={theme}>
      <App />
    </ThemeProvider>
  </React.StrictMode>,
  document.getElementById("root")
);

테마 사용

지정된 테마를 사용할 때는, ThemeProvider 컴포넌트로 감싸진 내부라면 어디서든 props.theme으로 접근할 수 있다.

1
2
3
4
5
6

import styled from "styled-components";

const Container = styled.div`
  background-color: ${(props) => props.theme.bgColor};
  color: ${(props) => props.theme.textColor};
`;
